# sligen

An slide generator that makes you pitch like a boss.

## How to use

1. Register an [Pixaby](https://pixabay.com/en/) account
2. Configure your src/crawlers/config.py
3. Run `src/download_image.py`
4. Use `make` to generate your slide

## Dependency

- nodejs
- python 3
    - beautifulsoup4
    - requests
    - jieba
